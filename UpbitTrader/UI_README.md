# 🚀 Upbit Auto Trader UI - 실행 가이드

## ✅ UI 프로토타입 완성!

업비트 거래소 스타일의 UI가 완성되었습니다!

## 📦 설치된 구성 요소

### 파일 구조
```
UpbitTrader/
├── main.py                  # 프로그램 진입점
├── requirements.txt         # 의존성 패키지
│
└── ui/
    ├── main_window.py       # 메인 윈도우
    ├── styles.py            # 다크 테마 스타일시트
    │
    └── widgets/
        ├── header_bar.py         # 상단 헤더 바
        ├── market_list.py        # 마켓 리스트 (좌측)
        ├── chart_widget.py       # 차트 (중앙)
        ├── orderbook_widget.py   # 호가/주문 (우측)
        └── bottom_tabs.py        # 하단 탭
```

## 🎨 구현된 UI 기능

### ✅ 완성된 부분
- [x] **헤더 바**
  - 로고 및 타이틀
  - 시스템 상태 표시
  - 자동매매 ON/OFF 토글
  - 계정 정보 (총 자산)
  - 설정 버튼

- [x] **좌측 사이드바 (마켓 리스트)**
  - 코인 검색 바
  - 필터 버튼 (전체/KRW/관심)
  - 마켓 리스트 테이블
  - 현재가 및 변동률 표시
  - 샘플 데이터 (BTC, ETH, XRP 등)

- [x] **중앙 차트 영역**
  - 시간 간격 선택 버튼
  - 지표 추가 버튼 (MA, RSI, MACD, AI 예측)
  - Matplotlib 차트 (샘플 데이터)
  - AI 예측 패널
    - 예측가 및 신뢰도
    - 매수/매도 시그널
    - 추천 전략 (진입가, 목표가, 손절가)

- [x] **우측 사이드바 (호가/주문)**
  - 호가창 탭
    - 매도/매수 호가 (색상 구분)
    - 현재가 하이라이트
  - 주문 탭
    - 매수/매도 폼
    - 가격/수량 입력
    - 수량 선택 버튼 (10%, 25%, 50%, 100%)
    - 매수/매도 버튼

- [x] **하단 탭 영역**
  - 보유 자산 (샘플 데이터)
  - 미체결 주문
  - 체결 내역 (샘플 데이터)
  - 거래 내역
  - AI 분석 (모델 성과 정보)

- [x] **메뉴 바**
  - 파일 메뉴 (설정, 로그, 종료)
  - 거래 메뉴 (자동매매 시작/중지, 긴급청산)
  - 전략 메뉴 (AI/RSI/복합 전략)
  - 분석 메뉴 (백테스팅, 성과분석)
  - 도움말 메뉴 (매뉴얼, 정보)

- [x] **스타일**
  - 업비트 스타일 다크 테마
  - 색상 코딩 (상승: 빨강, 하락: 파랑)
  - 반응형 레이아웃
  - 스플리터로 크기 조절 가능

## 🚀 실행 방법

### 1. 의존성 설치 (이미 완료)
```bash
pip install PyQt5 matplotlib numpy pandas
```

### 2. 프로그램 실행
```bash
python main.py
```

### 3. 종료
- 메뉴: 파일 → 종료
- 단축키: `Ctrl+Q`
- 윈도우 닫기 버튼 (X)
