# 🎉 실시간 차트 기능 완성!

## ✅ 구현 완료

### 새로 추가된 기능

1. **Upbit API 연동**
   - `pyupbit` 라이브러리 설치
   - `api/upbit_api.py` - API 래퍼 클래스
   - 실시간 마켓 데이터 가져오기
   - OHLCV 데이터 조회

2. **실시간 마켓 리스트**
   - 20개 코인 실시간 가격 표시
   - 5초마다 자동 업데이트
   - 변동률 색상 코딩 (빨강/파랑)
   - 검색 기능
   - 코인 클릭 시 차트 표시

3. **실시간 캔들 차트**
   - 업비트에서 실제 데이터 가져오기
   - 캔들스틱 차트 (빨강=상승, 청록=하락)
   - 시간 간격 선택 (1분, 5분, 15분, 1시간, 1일)
   - 현재가 표시
   - 새로고침 버튼

## 🚀 사용 방법

### 1. 프로그램 실행
```bash
python main.py
```

### 2. 코인 선택
- 좌측 마켓 리스트에서 원하는 코인 클릭
- 차트가 자동으로 해당 코인으로 변경됨

### 3. 시간 간격 변경
- 차트 상단 툴바에서 버튼 클릭:
  - 1분: 단기 스캘핑
  - 5분: 단기 트레이딩
  - 15분: 중기 트레이딩
  - 1시간: 장기 트레이딩
  - 1일: 투자 분석

### 4. 새로고침
- 🔄 새로고침 버튼으로 최신 데이터 로드
